# Summary

Ikea Lack table raiser for A8 i3 prusa.
Raises the table 10cm with holder for 5mm acrylic windows.
For the upper Part print Lack-Oben-Hinten two times (back side). Import Lack-Oben-Vorne,
copy it and mirror one of the parts (front side).
For the raiser print the Lack-Raiser two times (back side). Import Lack-Raiser-Front,
copy it and mirror one of the parts (front side).

The front window will be fitted with hinges.

It fits 3mm acryl, for the 5mm you have to file down the pin...

Added second version of upper parts with additional holes for the table legs (01.14.17)
